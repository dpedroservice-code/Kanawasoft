Visão Geral

O Kanawa Soft é um sistema de gestão empresarial completo, desenvolvido para atender micro, pequenas e médias empresas em Angola.

🛒

PDV

Ponto de venda completo com código de barras

📦

Estoque

Controle de produtos, lotes e validade

📊

BI

Business Intelligence e previsões

Instalação

1. Executável Windows

# Baixe o instalador\nwget https://kanawasoft.shop/download/KanawaSoft_Installer.exe\n# Execute o instalador\nKanawaSoft_Installer.exe

2. Docker (Produção)

# Clone o repositório\ngit clone https://github.com/kanawasoft/kanawa-soft.git\ncd kanawa-soft\n# Configure as variáveis\ncp .env.example .env\n# Inicie o sistema\ndocker-compose up -d

Primeiros Passos

1 Acesse o Sistema

Use as credenciais padrão: admin@kanawa.com / admin123

2 Configure a Empresa

Acesse Configurações → Empresa para definir nome, NIF e moeda

3 Cadastre Produtos

Vá em Produtos → Novo para adicionar seus produtos

4 Inicie as Vendas

Acesse PDV, abra o caixa e comece a vender!

PDV - Ponto de Venda

Funcionalidades

  • Leitura de código de barras
  • Venda com múltiplos itens
  • Desconto percentual e valor fixo
  • IVA configurável (14%, 7%, 0%)
  • Impressão automática de faturas
  • Clientes e vendedores

Atalhos de Teclado

F2Abrir PDV
EnterAdicionar produto
FFinalizar venda
CLimpar carrinho

Módulo Financeiro

  • Controle de contas a pagar e receber
  • Gestão de contas bancárias
  • Múltiplas formas de pagamento
  • Relatórios financeiros detalhados

Relatórios

  • Vendas por período
  • Produtos mais vendidos
  • Faturamento por vendedor
  • Análise de lucratividade
  • Exportação para PDF e Excel
